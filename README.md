# 22nd.kokasai.com
a Homepage of Kokasai, NITGC's school festival.

## Build

### Require

* npm
  * [harp](http://harpjs.com/) (`npm install -g harp`)
  

### Build step

1. `git clone https://github.com/eniehack/22nd.kokasai.com/`
1. `cd {cloned directory}`
2. `harp compile`
